Food Anywhere

A food delivery system that simplifies food ordering by connecting users with nearby restaurants. Built with Django, using PostGIS for geospatial data, Redis for caching and cart management, Celery for background tasks. The system is monitored with Prometheus, Grafana, and Loki. Pytest is used for testing, and Docker and Kubernetes for scalability and efficiency.

Job Hunt

A microservice-based job search platform that allows users to search for jobs based on their preferences. The users service is powered by FastAPI, while the jobs service is powered by Golang Fiber. The system is using Redis for caching and RabbitMQ for communication between services and kubernetes for deployment and github actions for CI/CD.
